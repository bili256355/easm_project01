# ROOT LOG APPEND — V10.7_g interpretation boundaries

## Forbidden interpretations

Do not interpret V10.7_g as causal evidence.

Do not treat self-target support, such as P→P45 or Jw→Jw41, as cross-object organization evidence. Self-target rows are diagnostic only.

Do not interpret missing object inputs as negative evidence. Missing objects must be treated as unavailable.

Do not claim the H negative result is decisive unless V10.7_g shows that the method can detect non-H cross-object signals.

## Route-use rule

If non-H sources show cross-target support while H does not, H can be downgraded more confidently for W45 main-cluster explanation.

If only self-target support or no support appears, the H negative result is not decisive; the method, metric, or window definition may be low-power.

If one source object shows clear cross-object support, follow-up spatial/high-low composites should be gated to that source-target pair only.
